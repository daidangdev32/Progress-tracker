$ Settings

GitHub Gist sync, data export, configuration

github gist sync

Free cloud backup using GitHub Gists. Your progress is stored as a JSON file in a secret gist that only you can access. Setup is ~2 minutes.

setup instructions (click to expand)
Step 1 — create a Personal Access Token
  1. Go to github.com/settings/tokens (fine-grained tokens)
  2. Click Generate new token
  3. Name it cybersec-tracker, expiration up to you
  4. Under Account permissions, set Gists: Read and write
  5. No repo access needed. Click Generate, copy the token (starts with github_pat_)
Step 2 — create a gist
  1. Go to gist.github.com
  2. Filename: cybersec-progress.json
  3. Content: {}
  4. Click Create secret gist
  5. From the URL gist.github.com/<you>/<gist-id>, copy the <gist-id> hex string
Step 3 — paste them below + click Connect

Token is stored only in your browser's localStorage. Never sent anywhere except api.github.com.
